# 🚀 Karan Bedi | Proof of Work Engine

A custom-built, dual-track portfolio showcasing my software engineering capabilities (AI, Data Analytics, Algorithmic Problem Solving) alongside my visionary concepts in Automotive Architecture.

🌐 **Live Site:** https://bedifutureworks.com

---

## 🧠 Philosophy

This is not a standard static resume.

It is a **Proof of Work Engine** designed to demonstrate:

- **Consistency** – Daily LeetCode architectures and GitHub commits.
- **Technical Depth** – Breakdowns of machine learning pipelines and database optimizations.
- **Systems Thinking** – Applying software logic to mechanical systems through Automotive Architecture research (e.g., hybrid torque vectoring and advanced differential systems).

---

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Deployment:** Vercel
- **Version Control:** Git & GitHub

---

## 📂 Project Structure

A clean component-driven architecture designed for scalability.

```
src/
├── app/                  # Next.js App Router (layout, page, globals.css)
├── components/
│   ├── layout/           # Global UI components (Navbar, Footer)
│   └── sections/         # Modular content sections
│       ├── ResumeSection
│       ├── CodeVaultSection
│       ├── ProjectsSection
│       ├── CreationsSection
│       ├── FutureBuildsSection
│       └── AutoArchitectureSection
└── data/                 # Static content arrays (skills, experience, projects)
```

## 🚀 Running Locally

To run this project on your local machine:

### 1. Clone the repository

```bash
git clone https://github.com/kbaylake/PersonalWebsite.git
cd PersonalWebsite/my-portfolio
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

in your browser.
